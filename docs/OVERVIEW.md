# MiniMax - Descripción General

MiniMax es una aplicación móvil de compras colaborativas pensada para conectar a pequeños emprendedores y comercios minoristas con proveedores mayoristas.

## Idea Central
Varios compradores pequeños pueden agrupar sus pedidos hasta alcanzar el mínimo exigido por un mayorista, accediendo así a mejores precios que individualmente no podrían conseguir.

## Tipos de Usuario

### 1. Comprador
- Explorar oportunidades de compra grupal.
- Sumarse a una oportunidad.
- Comprometer una cantidad.
- Seguimiento del progreso del grupo.

### 2. Proveedor
- Publicar oportunidades (producto, precio mayorista, cantidad mínima, fecha límite).
- Administrar el estado de cada publicación.
